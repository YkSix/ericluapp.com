MOS Order Android app

Play Store

This was an individual side project obtained and handled by myself from 2012 to 2014. With this app, users can pre-order food to skip the queues. This achieved two things: the users' food would be waiting for them when they arrived, as well as also help the restaurant become more efficient. More details are:

  • Created an O2O-featured Android app that assists in ordering meals conveniently.
  • Uses GCM push notification to receive the latest MOS Burger news
  • Location search feature using a power saving method.
  • Order meals using QR code (using zxing lib)
  • Check the recipe and ingredient of any MOS meals.

Take a look at this feature video to see MOS Order in action

Because I wanted to improve my Android development skill, I did some side projects after work, this was one of them. I was the only Android developer on this project.

The MOS Order Android app can be downloaded on Google Play Store.
